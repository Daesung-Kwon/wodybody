## 3. 🔧 기술적 이슈 및 대응

### 3.1 터치 이벤트 (마스킹)
- **구현**: `onTouchStart`, `onTouchMove`, `onTouchEnd`, `preventDefault()` 로 스크롤 방지
- **검증 필요**: iOS Safari에서 `touch-action: none` 과 `preventDefault` 동작 확인
- **대안**: 터치 영역 확대, 브러시 크기 조정

### 3.2 viewport·터치 영역
- **viewport**: `width=device-width, initial-scale=1.0` 설정됨
- **버튼**: MUI `minHeight: 44` 적용 (44px 이상 터치 권장)

### 3.3 입력 필드
- **체지방률**: `type="number"` + `step="0.1"` → 모바일에서 숫자 키패드 노출
- **날짜**: `type="date"` → 기기 기본 date picker 사용
