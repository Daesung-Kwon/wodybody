로컬 실행

홈에서 시작 버튼을 눌러 데모 흐름을 확인한 뒤, 저장소의 README를 참고해 로컬 환경을 준비하세요.

핵심 개념

운동 기록, 프로그램, 알림 3요소로 구성됩니다. 최소 설정으로 바로 기록을 시작할 수 있습니다.

FAQ

  • 데이터는 어디에 저장되나요?
    운영 환경에 따라 로컬 DB 또는 외부 DB를 사용할 수 있습니다.
  • 비밀번호 재설정은 지원하나요?
    예. 보안 토큰 기반으로 재설정을 지원합니다.

트러블슈팅

  • 로그인이 되지 않아요
    시간대 설정과 토큰 만료 시간을 확인하세요.
  • 알림이 오지 않아요
    이메일 또는 웹소켓 연결 상태를 점검하세요.
